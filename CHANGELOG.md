## ⚠️ About the Experimental Version (v0.1.x)

Currently, FerrumWard is in an **Experimental Version** phase.

- **What this means:** The core protection features, cryptographic license validation, and anti-tamper mechanisms are fully implemented and functional. However, the API boundaries, macro syntaxes, and internal architectures are subject to rapid changes based on community feedback.
- **Stability:** While highly effective against common attacks (e.g., memory injection, debuggers, time-freezing), this version is still gathering real-world field telemetry. It is highly recommended to thoroughly test FerrumWard in your game's staging environment before rolling it out to production.
- **Feedback:** We actively encourage developers to report any false positives, performance impacts, or compatibility issues (especially on specific Linux distributions or Proton versions). Please refer to our [Security Policy](SECURITY.md) for reporting channels.
